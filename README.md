
# Teste Frontend
A chatbot integrated with OpenAI
![devices](https://i.ibb.co/wWJcMhQ/3-devices-black.png)

# How to start
Install the dependencies by:
```
npm install
```
Install the dependencies by:
```
npm run mock
```
or 
```
npm run dev
```
that will need, the environment variables, as follow:
```
VITE_OPENAI_API_KEY=
VITE_OPENAI_API=https://api.openai.com/v1
```
